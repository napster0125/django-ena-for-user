django-userena
Accounts made beautifully simple

Userena is a Django application that supplies your Django project with full account management. It's a fully customizable application that takes care of the signup, activation, messaging and more. It's BSD licensed, which means you can use it commercially for free!

Got any feedback? Email me at petar@breadandpepper.com. For bugs you can report an issue at Github.

  • What is userena?

    Userena is an django application that supplies your project with full account management. It handles the signup, activation, signin etc. for you. It also supplies your users with an optional profile model that integrates gravatar avatars. Please signup at this website and try it out for yourself.

  • It's free!

    Userena is licensed under the BSD license. This means you it's also free for commercial use. You can adapt the software at free.

  • Used in production

    Bread & Pepper uses userena in their own project. Because of this we have revised a lot of the code and spared you of repeating this work. This has made us confident enough to be able to say that the the software is ready for production. Because it's open source, we also get feedback from the community that we use to build a solid product.

  • 100% test coverage

    Account management is an important aspect of any website, therefore we have chosen to have 100% test coverage for every bit of code. This increases the confidence that we can supply a great user experience.

  • Messaging

    Userena comes with an optional, chat-like, messaging system. Within this system users can see their communication with another user. It's use is intuitive and easy!

  • Privacy

    Permissions for viewing other peoples profiles are managed by django-guardian. This way we can let the user decide if they want to share their profile to everyone, registered users or nobody.
