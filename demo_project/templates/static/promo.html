django-userena
Accounts made beautifully simple

There are a lot of Django application that supply you with profiles, registration or accounts. We were tired of combining those to get full account functionality inside our applications. That's why we build userena. Full account management for your application, trying to make full use the current profile standards.

Try it out »

  • The usual..

    Of course userena has the standard tools that each account management software should have. Tools like signing in, up or out. Changing your settings, etc.

  • Gravatar

    Support for Gravatar. The user can upload it's own mugshot. If the user has no mugshot, userena is able to search a mugshot for the user by looking at the e-mail address.

  • Privacy

    userena want's you to be able to share only the things you want to share. So privacy options are a must. Select what to share to who.
